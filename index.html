Galeria Universitaria

Desarrollada con el framework Laravel 9. Un sistema de administracion de imagenes para la galeria universitaria que tiene la posibilidad de visualizar las imagenes en un entorno 3D diseñado a medidas.

Para este fin se utilizo el gameengine Godot en su version 3.4.4, el cual entre sus funcionalidades permite realizar peticiones HTTP a un servidor remoto para obtener las imagenes y datos de las mismas.
